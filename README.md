# Quasar Framework

> 以创纪录的速度构建高性能的 Vue.js 用户界面：响应式单页应用（SPA）、服务端渲染应用（SSR）、静态站点生成应用（SSG）、渐进式 Web 应用（PWA）、浏览器扩展、混合移动应用以及 Electron 桌面应用。如果你愿意，这一切都可以使用同一套代码库完成！

## 支持 Quasar

Quasar Framework 是一个基于 MIT 许可证的开源项目。它能够持续开发，离不开我们出色的赞助商和支持者的支持。

也请考虑支持我们的工作！如果你在工作中使用 Quasar，请向你的管理层提议在 https://donate.quasar.dev/ 上赞助我们。另外也请阅读[为什么捐赠很重要](https://quasar.dev/why-donate)。
